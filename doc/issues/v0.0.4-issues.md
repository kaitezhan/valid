
# message

允许 Constraint() 自定义 message 信息。

```java
ValidBs.on("value", constraintOne.message("XXX"))
```

# 约束链

对同一个字段，允许多个约束条件。

定义 chainInit()，或者

```java
ValidBs.on("value", constraintOne, ...)
```

## 接口的规范性

直接使用

```java
ValidBs.on("value", newConstraintInit() {
    // 真正的验证是遍历列表中的验证信息。
    pipeline.add();
})
```
