# IResult 新增两种 Handler 方法。

## 处理模式

- print()

输出到控台，所由校验信息。

后期考虑支持输出信息源头。

- throws

抛出异常。

如果存在 notPass 的情况，直接抛出 ValidRuntimeException



# onEach() 特性支持

支持 Iterable 的属性。

（1）集合

（2）数组

（3）Map

也可以只添加一个 util 工具类。

```java
// 返回一个可遍历的 list
EachUtil.each(Object)
```
