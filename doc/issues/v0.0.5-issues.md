# 支持单个字段，指定多个约束条件

## 支持完整的 builder

和 ValidEntry 比就是没有 value 这个属性。

```java
ConstraintEntryBuilder
.constraint("XXX")
.when()
.message()
.group()
```

## 指定多个 constraint 示例

```java
ValidBs.on(value, ConstraintEntryOne, ConstraintEntryTwo, ...)
```

这些约束之间互相独立。

（2）验证时
